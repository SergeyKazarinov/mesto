# Проект: Место

### Обзор
Интерактивной страница, куда можно добавлять фотографии, удалять их и ставить лайки.

* Figma
* Файловая структура организована по БЭМу.
* Вёрстка адаптивная. Минимальная ширина: 320px. Максимальная: 1280px.
* Pop-up
